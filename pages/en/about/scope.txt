====== XBUP Protocol Scope ======

Following content should be created as a part of ExBin - XBUP project:

  * Protocol specification
  * Software implementation
    * Parsers (Interfaces for acessing protocol in it's basic structure)
      * Document object model parser
      * Event/pull parser
      * Command parser
    * Support for serialization
    * Support for streaming
    * Implementation of catalog as service
    * Transformation service
    * Universal editor tool
    * Catalog type web service
  * Documentation
  * Example files
